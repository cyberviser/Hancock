Frequently Asked Questions

Do I need a GPU to run Hancock?
Not for inference. The Community plan uses NVIDIA NIM cloud inference — you just need an NVIDIA API key (free tier available at build.nvidia.com). For local fine-tuning, a GPU with 16GB+ VRAM is recommended.
Can I use Hancock in commercial pentests?
Yes — with a Pro or Enterprise license. The Community plan is for personal/research use only. Commercial use requires a signed license agreement.
Is Hancock safe to use in a production SOC?
The Enterprise plan includes on-premise deployment, so your alerts and data never leave your environment. Pro plan traffic is encrypted in transit and at rest. We do not log conversation content.
What SIEM platforms does Hancock integrate with?
Hancock generates native queries for Splunk SPL, Elastic KQL, and Microsoft Sentinel KQL out of the box via the /v1/hunt endpoint. Direct integrations (Splunk app, Elastic plugin) are on the roadmap.
How does billing work?
Pro is billed monthly per seat. Usage-based billing is metered per API request. Enterprise is annual contract. All plans can be cancelled anytime — no lock-in.
